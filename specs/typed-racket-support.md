## Features from SML that are supported so far:

### Types and Types Constructors

Base types:

* `Fixnum` (64 bit 2's complement)

* `Flonum` (IEEE 64)

* `Boolean` (`#t` or `#f`)

Type constructors: 

* `t1 * t2` (Tuple)

* `-> t1 t2` (Function type)

* `Listof t` (Homogeneous lists)

* `All (a1 ... an) t` (Parametric polymorphism)

### Operations

* Arithmetic: `+`, `-`, `*`

* Boolean-valued: `=`, `<`, `>`, `<=`, `>=`

### Control

* `if e1 e2 e3` Ummm
